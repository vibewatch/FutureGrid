## Notable Issue: BarChartRace jsdom/d3 Transition Cleanup

**Problem:** BarChartRace had no cleanup in its d3 useEffect. After unmount, d3's rAF callbacks attempt to interpolate `transform` attributes via `d3-interpolate`'s `parseSvg()`, which calls `SVGElement.transform.baseVal.consolidate()`. jsdom does not implement `SVGElement.transform` (undefined), causing unhandled TypeError exceptions post-teardown.

**Resolution:** 
- Test: `beforeAll` polyfills `SVGElement.prototype.transform` with `{ baseVal: { consolidate: () => null } }`
- **Component fix applied (commit 6a6160b):** Added cleanup to BarChartRace's d3 useEffect: `return () => { d3.select(svgEl).selectAll("*").interrupt(); };`
